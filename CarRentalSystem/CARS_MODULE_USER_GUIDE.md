## ?? Обзор модуля

Модуль "Автомобили" - это центральный компонент системы проката автомобилей, который позволяет:
- Управлять автопарком компании
- Отслеживать статус каждого автомобиля
- Просматривать историю аренд и ремонтов
- Контролировать страхование
- Устанавливать цены на аренду

### Доступ к модулю:
- В главном меню: **Vehicles** ? **All Cars**
- С Dashboard: кнопка **View Cars**

## ?? Фильтрация и поиск

### Панель фильтров

#### 1. **Текстовый поиск**
Найдите автомобиль по:
- Регистрационному номеру
- Локации
- Названию модели
- Марке автомобиля

**Пример:** Введите "Toyota" для поиска всех Toyota

#### 2. **Фильтр по марке**
Выберите марку из выпадающего списка:
- Показываются только автомобили выбранной марки
- Выберите "All Brands" для сброса фильтра

#### 3. **Фильтр по модели**
Выберите конкретную модель:
- Работает вместе с фильтром по марке
- Выберите "All Models" для сброса

#### 4. **Фильтр по статусу**
Фильтруйте по доступности:
- Available (Доступен)
- Rented (Арендован)
- Reserved (Зарезервирован)
- Maintenance (На обслуживании)

**Применение:** Быстро найдите все доступные автомобили

#### 5. **Диапазон цен**
Укажите минимальную и максимальную дневную цену:
- **Min:** минимальная цена (например, 50)
- **Max:** максимальная цена (например, 200)

**Пример:** Найдите автомобили от $50 до $150 в день

### Применение фильтров:
1. Настройте нужные фильтры
2. Нажмите **Apply Filters**
3. Для сброса нажмите **Clear Filters**

**Совет:** Фильтры можно комбинировать для точного поиска!

**Информация о марке:**

## ? Добавление нового автомобиля

Нажмите кнопку **Add New Car** на главной странице.

### Заполнение формы:

#### **Раздел 1: Vehicle Identification (Идентификация)**

1. **Brand (Марка)** *
   - Выберите марку из списка
   - Обязательное поле

2. **Model (Модель)** *
   - После выбора марки автоматически загружаются её модели
   - Выберите модель из списка
   - Обязательное поле

3. **Registration Number (Регистрационный номер)** *
   - Введите уникальный регистрационный номер
   - Максимум 10 символов
   - Автоматически преобразуется в верхний регистр
   - Пример: ABC-1234
   - Должен быть уникальным

#### **Раздел 2: Vehicle Details (Детали)**

4. **Mileage (Пробег в км)** *
   - Текущий пробег автомобиля
   - Только положительные числа
   - Пример: 50000

5. **Daily Rental Price (Дневная цена аренды)** *
   - Стоимость аренды за один день
   - Положительное число
   - Пример: 89.99

6. **Availability Status (Статус доступности)** *
   - Available - доступен
   - Rented - арендован
   - Reserved - зарезервирован
   - Maintenance - на обслуживании

7. **Location (Локация)**
   - Необязательное поле
   - Место нахождения автомобиля
   - Пример: Main Office, Branch #2

### Сохранение:
- **Add Car** - сохранить новый автомобиль
- **Cancel** - отменить и вернуться к списку

### Важно:
- Поля со звёздочкой (*) обязательны
- Сначала выберите марку, затем модель
- Регистрационный номер должен быть уникальным

## ?? Редактирование автомобиля

Нажмите **Edit** на карточке или странице деталей.

### Процесс редактирования:

1. **Форма предзаполнена** текущими данными
2. Измените нужные поля
3. При изменении марки модели обновятся автоматически
4. Нажмите **Update Car** для сохранения
5. Или **Cancel** для отмены

### Что можно изменить:
- Марку и модель
- Регистрационный номер (с проверкой уникальности)
- Пробег (можно только увеличивать)
- Дневную цену
- Статус доступности
- Локацию

### Что нельзя изменить:
- ID автомобиля (отображается для справки)

### Дополнительные действия:
- **View Details** - просмотреть полную информацию
- **Cancel** - отменить изменения

## ??? Удаление автомобиля

Нажмите **Delete** на карточке или странице деталей.

### Процесс удаления:

#### **Если автомобиль БЕЗ аренд:**

1. Отображается информация об автомобиле
2. Показывается, что будет удалено:
   - Сам автомобиль
   - Связанная страховка (если есть)
   - Все записи о ремонтах
3. Требуется подтверждение
4. Нажмите **Confirm Delete**
5. Подтвердите в диалоговом окне

**Результат:** Автомобиль и все связанные данные удалены

#### **Если автомобиль С арендами:**

? **Удаление НЕВОЗМОЖНО!**

Отображается предупреждение:
- Количество аренд в системе
- Причина блокировки удаления
- Рекомендация сохранить для истории

**Почему нельзя удалить?**
- Сохранение истории аренд
- Целостность данных
- Аудит и отчётность

### Альтернативы удалению:
1. Измените статус на **Maintenance**
2. Обновите информацию вместо удаления
3. Свяжитесь с администратором

## ?? Частые ошибки и решения

### Проблема: "A car with this registration number already exists"
**Решение:** Выберите другой уникальный номер

### Проблема: Не загружаются модели
**Решение:** Сначала выберите марку

### Проблема: "Cannot delete car"
**Решение:** У автомобиля есть история аренд - удаление невозможно

### Проблема: Фильтры не работают
**Решение:** Нажмите "Apply Filters" после настройки
